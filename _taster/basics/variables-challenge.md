## Variables Challenge

Using the following variables

	greeting = "Hello!"
	introduction = "My name is"
	name = "Alice"
	luckyNumberText = "My lucky number is"
	luckyNumber = 13


See if you can write a program which produces

	Hello!
	My name is Alice
	My lucky number is 13
